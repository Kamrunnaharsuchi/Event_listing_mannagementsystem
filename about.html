    About Our Project

    Welcome to our University Event Listing Management System, a project designed to showcase the vibrant activities and events happening within our university.

    This platform provides visitors with an engaging and informative experience,
    allowing them to explore upcoming events, view detailed event pages,
    and discover related activities happening on campus.

    We’ve developed a fully dynamic landing page and event details page using HTML, CSS, and JavaScript.
    All event data is stored and managed through a JSON file, eliminating the need for a traditional database.
    The backend is powered by Node.js, ensuring seamless data handling,
    while the frontend offers a smooth and responsive user experience.

    To streamline event management, we’ve also built an intuitive admin panel,
    enabling easy control over event listings.
    This allows our university to keep the platform updated with the latest happenings.

    Through the admin panel, we offer complete control over event listing,
    gallery management, category handling, user authentication,
    and support for multiple event post types including Normal, New, and Upcoming.

    This project is proudly developed by our team, Flairia,
    as part of our university initiative to create impactful and user-friendly digital solutions.

    This is a dynamic University Event Listing System.
    Showcases upcoming events and campus activities.
    Built with HTML, CSS, JavaScript, and Node.js.
    Event data is managed using a JSON file backend.
